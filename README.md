# extracta
A web app made for people who don't know how to use excel.

## How to run
- Requirements:
    - Python 3.1+
    - Node.js

- For Mac (in Terminal):
1. Navigate to "extracta" root folder in local computer
2. Run "chmod +x run.sh"
3. Run "./run.sh"
